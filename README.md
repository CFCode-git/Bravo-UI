## 开始使用

1. 添加CSS样式

使用本框架前，注意CSS样式中使用border-box。
```$xslt
*{box-sizing: border-box;}
*::before{box-sizing: border-box;}
*::after{box-sizing: border-box;}
```
此样式支持 IE 8 及以上浏览器。
目前还需您手动设置默认颜色变量。

```
html {
  --button-height: 32px;
  --font-size: 14px;
  --button-bg: white;
  --button-active-bg: #eee;
  --border-radius: 4px;
  --color: #333;
  --border-color: #999;
  --border-color-hover: #666;
}
```

2. 安装
```$xslt
npm i --save bravo-ui
```
